## 2. 프로젝트 설치 방법
a. 터미널을 이용하여 로컬에 설치할 경로로 이동해줍니다.

```
$ cd 설치할 경로
```

b. 아래의 코드를 복사 후 붙여넣어, 코드를 모두 다운로드 해줍니다.

```
$ git clone git@github.com:swliris/OSS_Final_Proj_T2.git
```

c. 해당 설치 경로에 'OSS_Final_Proj_T2' 라는 이름의 폴더가 생겼다면, 설치에 성공한 것 입니다!
